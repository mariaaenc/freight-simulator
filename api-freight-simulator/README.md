# api-freight-simulator

## Description

API responsible for the context of freight and logistics operators.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

```bash
$ heroku container:push web

$ heroku container:release web

# container logs
$ heroku logs --tail
```
